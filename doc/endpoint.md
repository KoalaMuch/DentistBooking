# Endpoint

This is endpoints for dentist booking application

## User `/auth`

### Register `/register`

register new user to system

### Login `/login`

set token cookie to user's request

### User's info `/me`

get basic information of logged in user

### Logout `/logout`

delete token cookie to user's request

## Clinic `/clinic`

### Get all `/`

get all clinics
